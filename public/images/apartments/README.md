
# Cartelle per le immagini degli appartamenti

In questa directory sono presenti le cartelle per ciascun appartamento:

- `appartamento-1/`: Contiene le immagini dell'Appartamento 1
- `appartamento-2/`: Contiene le immagini dell'Appartamento 2
- `appartamento-3/`: Contiene le immagini dell'Appartamento 3
- `appartamento-4/`: Contiene le immagini dell'Appartamento 4

## Struttura delle immagini

Per ogni appartamento, caricare le immagini seguendo questa convenzione:
- `image1.jpg`: Immagine principale (copertina)
- `image2.jpg`, `image3.jpg`, ecc: Immagini aggiuntive

L'immagine denominata `image1.jpg` verr√† utilizzata automaticamente come immagine di copertina nella visualizzazione dell'appartamento.

## Esempio
Per l'Appartamento 1:
```
/public/images/apartments/appartamento-1/image1.jpg  # Copertina
/public/images/apartments/appartamento-1/image2.jpg
/public/images/apartments/appartamento-1/image3.jpg
```

Per caricare nuove immagini tramite GitHub, naviga alla cartella corrispondente all'appartamento desiderato e utilizza la funzione "Upload files".
